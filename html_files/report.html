Introduction
What is a Scalar Field?
Suppose we want to measure the temperature at of the surface of the Earth over a large region. We can think of this region as simply being a two-manifold ignoring the possible changes in height as we sample and only taking into consideration the geographic coordinates. Denote this space as $\mathbb{X}$. Then we also have a function $f:\mathbb{X}\to \mathbb{R}$ mapping from this two-manifold to the real numbers by associating the coordinate pair with the temperature at this point. The pair $(\mathbb{X},f)$ is called a scalar field. We can often visualize well-behaved scalar fields as some sort of mountainous or hilly terrain. Just as we can use contour lines on a map to indicate regions that are the same elevation, we are sometimes interested in a subset $U \in \mathbb{X}$ such that all points in $U$ have the same function value. These are called level sets or fibers of the scalar field Maybe something about our concern with the evolution of the level sets
Comparing Scalar Fields
Topological Evolution of a Scalar Field
Let $(\mathbb{X},f)$ be a scalar field. We can define an equivalence relation on $\mathbb{X}$ by defining $x\sim y$ if $f(x) = f(y) = a$ (they belong to the same $a$-fiber) and if $x,y$ are in the same path connected component of the $a$-fiber. Then, the quotient space $\mathbb{X}_f := \mathbb{X}/\sim_f$ is called the Reeb Graph of the scalar field $(\mathbb{X},f)$. We equip the Reeb Graph with the function $\tilde{f}$ which is defined as $\tilde{f} \circ \rho = f$, where $\rho: \mathbb{X} \to \mathbb{X}_f$ is the quotient map defined by our equivalence relation. The Reeb Graph tracks the evolution of the levelset topology of the scalar field. It shows how the fibers of the scalar field split and merge.

Category Theory
A category is a collection $\textbf{C}$ of objects with morphisms (also called maps) that map between these objects. The set of objects in $\textbf{C}$ is denoted as $\mathsf{ob}(\textbf{C})$ and the set of morphisms is denoted as $\mathsf{hom}(\textbf{C})$. When reffering to an object $A$ in a category $\textbf{C}$, we often denote this as $A \in \textbf{C}$ instead of the more precise notation of $A \in \mathsf{ob}(\textbf{C})$ when it is clear from the context that $A$ is an object and not a morphism. Each morphism $\alpha$ has a source $A\in \textbf{C}$ and a target $B\in \textbf{C}$. Morphisms usually represent some sort of "structure" preserving idea, such as isomorphism when considering the category of vector spaces $\textbf{Vect}$, or homomorphisms in the category of groups $\textbf{Grp}$. Every object $A$ in a category $\textbf{C}$ has an identity morphsim $\textbf{id}_A:A \to A$.
A functor $\afunc{F}$ is a mapping between two categories $\textbf{C}$ and $\textbf{D}$ that satisfies the following properties:
  • For each object $A \in \textbf{C}$, there is a corresponding object $\afunc{F}(A) \in \textbf{D}$
  • For each moprhism $\alpha: A \to B$ in $\textbf{C}$, there is a corresponding morphism $\afunc{F}[\alpha]:\afunc{F}(A) \to \afunc{F}(B)$ in $\textbf{D}$.
  • The functors respect composition: $\afunc{F}[\alpha \circ \beta] = \afunc{F}[\alpha] \circ \afunc{F}[\beta]$
  • The functors respect identities: $\afunc{F}[\textbf{id}_A] = \textbf{id}_{\afunc{F}[A]}$
A functor can be thought of as a way to map between categories in the same way that functions map between spaces (vector spaces, groups, etc.)

Let $\textbf{C}$ be a category. A subcategory $\textbf{S}$ of $\textbf{C}$ is a category consisting of a subcollection of objects of $\textbf{C}$, denoted as $\mathsf{ob}(\textbf{S})$ and a subcollection of morphisms of C, denoted as $\mathsf{hom}(\textbf{S})$, such that
  • for every $X$ in $\mathsf{ob}(\textbf{S})$, the identity morphism $\textbf{id}_X$ is in $\mathsf{hom}(\textbf{S})$
  • for every morphism $\alpha: X \to Y$ in $\mathsf{hom}(\textbf{S})$, both the source $X$ and the target $Y$ are in $\mathsf{ob}(\textbf{S})$
  • for every pair of morphisms $\alpha$ and $\beta$ in $\mathsf{hom}(\textbf{S})$, the composite $\alpha \circ \beta$ is in $\mathsf{hom}(\textbf{S})$ (whenever the composition is defined).
A class is any collection of objects and a proper class is any collection of objects that is not a set. A proper class is a collection of objects which contradicts one of the axioms of Zermeloâ€“Fraenkel set theory. Informally, we can think of a proper class to be a collection which is 'too big' to be a set itself. For example, consider the collection $\mathsf{S}$ of all sets. Then, every subcollection of $\mathsf{S}$ is also a set and therefore is in $\mathsf{S}$. If we assume $\mathsf{S}$ is a set, then every element of the power set $P(\mathsf{S})$ of $\mathsf{S}$ is also in $\mathsf{S}$. This means that $P(\mathsf{S}) \subseteq \mathsf{S}$ which is a contradiction of Cantor's Theorem which states that the cardinality of the power set is strictly larger than the set itself.
Examples of proper classes include the collection of all sets, the collection of all topological spaces, and the collection of all vector spaces. To see this, note that any set can be given the discrete topology. So the collection of all topological spaces includes teh collection of all sets equipped with the discrete topology. Since the collection of all sets is not a set, the collection of all topological spaces is not a set. A similar argument can be made for the collection of all vector spaces.

Category of Scalar Fields
We can construct the category of scalar fields (in related work also called the category of $\mathbb{R}$-spaces) denoted as $\rtop$ by stating that each object is a scalar field $(\mathbb{X},f)$ and each morphism $\alpha: (\mathbb{X},f) \to (\mathbb{Y},g)$ is a continuous, function-preserving map $\alpha:\mathbb{X}\to \mathbb{Y}$ between topological spaces. A function preserving map means that $\alpha \circ g = f$. In other words, the following diagram commutes:
We call the point pre-images $f^{-1}(a)$ of $\mathbb{X}$ the $a$-levelset or $a$-fiber of $\mathbb{X}$. In this way, we can say that the morphisms in the category of scalar fields preserve $a$-levelsets or $a$-fibers.
To make sure that we are dealing with well-behaved scalar fields, we define a new subcategory called the category of constructible scalar fields, denoted as $\rtopc$.
Let $S = \{a_1,\ldots,a_n\}$ be a finite set of real numbers, called "critical points". We define a critical fiber, $\mathbb{V}_i$, for each $0\leq i \leq n$ and a non-critical fiber, $\mathbb{E}_i$ for each $0 \leq i \leq n-1$. Each critical and non-critical fiber is defined to a be a locally path-connected, compact space.
Now we define continuous attaching maps $l_i:\mathbb{E}_i \to \mathbb{V}_i$ and $r_i:\mathbb{E}_i \to \mathbb{V}_{i-1}$ for each $0 \leq i \leq n-1$. Finally, we define $\hat{\mathbb{X}}$ to be the quotient space obtained by the disjoint union of $\mathbb{V}_i \times \{a_i\}$ and $\mathbb{E}_i \times [a_i,a_{i+1}]$ by making the identifications $(l_i(x),a_i) \sim (x,a_i)$ and $(r_i(x),a_{i+1}) \sim (x,a_{i+1})$ for all $i$ and all $x \in \mathbb{E}_i$. Now, define a function $f$ such that $f(x,a) = a$. We can see that the pair $(\hat{\mathbb{X}},f)$ defines a scalar field since $\hat{\mathbb{X}}$ is a topological space and $\hat{f}$ is simply a projection onto the second factor, which is a continuous function.
If $\mathbb{X}$ is isomoprhic to $\hat{\mathbb{X}}$, then $\mathbb{X}$ is a constructible scalar field and we say that $\hat{\mathbb{X}}$ is its constructible base. Note that the choice of constructible base is not unique But I believe all are homotopic. The intuition here is that the scalar fields are "well-behaved" if they have a finite set of critical points and if between critical points, their shap is homotopic to a cylinder. Then at the critical fibers, we have places where the cylindrical structures are able to "merge" or "split". Note again that these attaching maps are always mapping from the non-critical fibers. So if our non-critical fiber has a cylinder that split into two cylinders, there must be a critical fiber which is a figure eight, exactly at the point where they start to split. We can always map a single cylinder to a figure eight or two cylinders to the figure eight. We never have to worry about actually "splitting" the figure eight into two cylinders (which is discontinuous) because the maps only go towards the critical-fibers.

The set of Reeb Graphs created from scalar fields defines a subcategory of $\rtop$.
To prove this, we need to show that the Reeb Graphs are scalar fields and that there exists a subcollection of morphisms of $\mathsf{hom}(\rtopc)$ where the source and target are always Reeb Graphs. To help us with the following proofs, we refer to the lemma below:
(from Munkres, Theorem 22.2) Let $\rho: X \to Y$ be a quotient map. Let $Z$ be a space and let $f: Y \to Z$ be a map that is constant on each set $\rho^{-1}(\{y\})$, for $y \in Y$. Then $f$ induces a map $g:Y \to Z$ such that $g \circ \rho = f$. The induced map $g$ is continuous if and only if $f$ is continuous.
Let $(\mathbb{X},f)$ be a scalar field and $(\mathbb{X}_f, \tilde{f})$ denote its Reeb Graph. Then $(\mathbb{X}_f, \tilde{f})$ is a scalar field.
By definition $\mathbb{X}_f$ is a topological space. What is left to show is that $\tilde{f}$ is a continuous function. Let $\rho$ be the quotient map from $\mathbb{X}$ to $\mathbb{X}_f$. Since $f$ is constant on point pre-images of $\mathbb{X}_f$, Munkres Theorem 22.2 tells us that there exists a continuous function $g \circ \rho = f$. By definition of a Reeb Graph, we have that $\tilde{f} \circ \rho = f$, so $\tilde{f} = g$ and $(\mathbb{X}_f, \tilde{f})$ is a scalar field.
Let $(\mathbb{X},f)$ and $(\mathbb{Y},g)$ be scalar fields, let $(\mathbb{X}_f,\tilde{f})$ and $(\mathbb{Y}_g,\tilde{g})$ be their Reeb Graphs resepectively, and let $\rho_f$ and $\rho_g$ be the quotient maps from the scalar fields to the Reeb Graphs. If $\alpha$ is a morphism between $(\mathbb{X},f)$ and $(\mathbb{Y},g)$, then the map $\tilde{\alpha}$ defined by $\rho_g \circ \alpha = \tilde{\alpha} \circ \rho_f$ is a morphism in $\rtop$.
Consider the composition of maps $(\rho_g \circ \alpha): \mathbb{X} \to \mathbb{Y}_g$. This map is continuous since both $\rho_g$ and $\alpha$ are continuous. Now, consider a point pre-image $\rho_f^{-1}(\{y\}) \in \mathbb{X}$ where $\tilde{f}(y) = a$. Then this pre-image is the $a$-fiber of $\mathbb{X}$. Since $\alpha \in \mathsf{hom}(\rtop)$, it preserves level sets, so $\alpha(\rho_f^{-1}(\{y\}))$ is the $a$-fiber of $\mathbb{Y}$. Then, since $\rho_g$ is the quotient map turning all $a$-fibers into a single contracted point in $\mathbb{Y}_g$, the composition of maps $(\rho_g \circ \alpha)$ is constant on pre-images of $\rho_f$. Using Munkres Theorem 22.2 again, we have that the induced map $\tilde{\alpha}$ in the equation $\rho_g \circ \alpha = \tilde{\alpha} \circ \rho_f$ must be continuous, since $\rho_g \circ \alpha$ is continuous and constant on the point pre-images of the quotient map $\rho_f$.
To show that $\tilde{\alpha}$ is a function preserving map, we need to show that $\tilde{f} = \tilde{g} \circ \tilde{\alpha}$. We begin with the fact that $f = g \circ \alpha$: \begin{equation} f = g \circ \alpha \hspace{0.1in} \Rightarrow (\tilde{f} \circ \rho_f) = (\tilde{g} \circ \rho_g) \circ \alpha \hspace{0.1in} \Rightarrow (\tilde{f} \circ \rho_f) = \tilde{g} \circ (\rho_g \circ \alpha) \hspace{0.1in} \Rightarrow (\tilde{f} \circ \rho_f) = \tilde{g} \circ (\tilde{\alpha} \circ \rho_f) \hspace{0.1in} \Rightarrow \tilde{f} = \tilde{g} \circ \tilde{\alpha} \end{equation}

Pre-Cosheafs
Pre-cosheafs allow us to abstract the data the a Reeb Graph provides us. In this section, we define what a pre-cosheaf is and show how we can describe the pre-cosheaf of a Reeb Graph.
Let $\textbf{Open}(\mathcal{X})$ be the cateogry of open sets on the topological space $\mathcal{X}$ where $I \to J$ if $I \subseteq J$. Then a pre-cosheaf is a functor $\mathsf{F}: \textbf{Open}(\mathcal{X}) \to \textbf{D}$ where $D$ is some category. In other words, a pre-cosheaf is an assignment of data to open sets of a topological space $\mathcal{X}$.
In this work, our pre-cosheaves will exclusively have the category of intervals on the real line $\textbf{Int}$ as the domain space and the category of sets $\mathbf{Set}$ as the range space. Note that the morphisms in $\textbf{Set}$ are functions between sets.
We define the category of pre-cosheaves $\textbf{Pre}$ as functors from $\textbf{Int}$ to $\textbf{Set}$. In other words, $\textbf{Pre} := \textbf{Set}^{\textbf{Int}}$
Let $\textbf{Int}$ denote the category of intervals $I\subseteq \mathbb{R}$ where the morphisms between the intervals is inclusion. We usually denote this morphism using the subest notation $I \subseteq J$.
Let $\textbf{Set}$ note the category of sets where the morphisms between objects are functions between the sets.
A Reeb Graph naturally encodes data about the connected components of the level sets of a scalar field. So, it makes sense to be able to view this as a pre-cosheaf where our domain space are the intervals of the real line and the data attached to these intervals is a set of path-connected components.
Let $f = (\mathbb{X},f)$ be a scalar Reeb Graph. The Reeb cosehaf functor $\mathcal{C}$ is defined by the formulas \[\mathsf{F}(I) = \pi_0(f^{-1}(I)), \hspace{0.5in} \mathsf{F}[I\subseteq J] = \pi_0[f^{-1}(I) \subseteq f^{-1}(J)].\] This functor converts a Reeb Graph to its pre-cosheaf. Thus, $\mathcal{C}: \textbf{Reeb} \to \textbf{Pre}$.
Interleaving Distance
Interleaving of Pre-Cosheafs
The interleaving distance between two pre-cosheaves is a way to gauge how close they are to each other based on "approximate isomorphisms".
We say that two pre-cosheafs $\mathsf{F},\mathsf{G}$ are $\varepsilon$-interleaved if there exists a family of morphisms \[\varphi_I:\mathsf{F}(I) \to \mathsf{G}(I^{\varepsilon}), \hspace{0.75in} \psi_I:\mathsf{G}(I) \to \mathsf{F}(I^{\varepsilon}),\] which are natural with respect to the inclusions $I \subseteq J$, such that \[\psi_{I^{\varepsilon}} \circ \varphi_I = \mathsf{F}[I\subseteq I^{2\varepsilon}], \hspace{0.75in} \varphi_{I^{\varepsilon}} \circ \psi_I = \mathsf{G}[I\subseteq I^{2\varepsilon}]\] Note that when $\varepsilon = 0$, this is exactly an isomorphism between $\mathsf{F}$ and $\mathsf{G}$. When two pre-cosheafs are $\varepsilon$-interleaved, we say that there exists an $\varepsilon$-interleaving between them.
The interleaving distance between two pre-cosheafs $\mathsf{F}$ and $\mathsf{G}$ is defined as \[ d_I(\mathsf{F},\mathsf{G}) = \inf\{\varepsilon \geq 0 | \text{ there exists an } \varepsilon \text{-interleaving between } \mathsf{F},\mathsf{G}\}\]
Interleaving of Reeb Graphs
Let $f = (\mathbb{X},f)$ and $g = (\mathbb{Y},g)$ be two Reeb Graphs. Since we have a notion of distance between two pre-cosheafs, we can define the interleaving distance between $f,g$ simply by \[d_I(f,g) := d_I(\mathcal{C}(f),\mathcal{C}(g))\]
Unfortunately, this definition of the interleaving distance is very abstract and requires converting our Reeb Graphs to pre-cosheafs before defining the distance. What we seek to find is a nice geometric realization of this distance.
Smoothing Functors
Even categorically, the interleaving distance between two cosheafs is difficult to describe. Here, we introduce the idea of the Smoothing Functor. Our goal is to be able to construct a map between pre-cosheafs which encompasses the notion of $\varepsilon$-interleaving.
The expansion functor by $\varepsilon > 0$, $\Omega_{\varepsilon}$, is described as \[\Omega_{\varepsilon}: \textbf{Int} \to \textbf{Int} ;\hspace{0.1in} I \mapsto I^{\varepsilon}\]
This is a well defined functor since when $I \subseteq J$ we have $I^{\varepsilon} \subseteq J^{\varepsilon}$. Thickening Functors
Here we describe a thickening functor on Reeb Graphs that operates in parallel to the smoothing functors described previously. What we will show is that it is equivalent to work with thickened Reeb Graphs or smoothed pre-cosheafs.
For $\varepsilon \geq 0$, we define the thickening functor $\mathcal{T}_{\varepsilon}:\rtop \to \rtop$ as follows:
  • Let $(\mathbb{X},f)$ be a scalar field, let $\mathbb{X}_{\varepsilon} = \mathbb{X} \times [-\varepsilon,\varepsilon]$, and let $f_{\varepsilon}(x,t) = f(x) + t$. Then $\mathcal{T}_{\varepsilon}(\mathbb{X},f) = (\mathbb{X}_{\varepsilon},f_{\varepsilon})$
  • Let $\alpha:(\mathbb{X},f) \to (\mathbb{Y},g)$ be a morphism in $\rtop$. Then we define $\mathcal{T}_{\varepsilon} : (\mathbb{X}_{\varepsilon},f_{\varepsilon}) \to (\mathbb{Y}_{\varepsilon},g_{\varepsilon})$ by $(x,t) \mapsto (\alpha(x),t)$.
The functors $\mathcal{C}\mathcal{T}_{\varepsilon}$ and $\mathcal{S}_{\varepsilon}\mathcal{C}$ are naturally isomorphic.
