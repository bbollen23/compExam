The Category of Scalar Fields
Our goal in this section is to define some preliminary definitions and content. We hope to give a general overview of the cateory theoretic approach, while not getting too invovled in details that do not immediately concern the scalar fields that we are looking into.
Suppose we have a topological space $X$ equipped with a continuous map $f : X \rightarrow \mathbb{R}$. We call this a scalar field and denote it as $(X,f)$.
The category of scalar fields is denoted $\mathbb{R}-\textbf{Top}$ and a morphism between scalar fields $\varphi: (X,f) \rightarrow (Y,g)$ is a continuous map $\varphi :X \rightarrow Y$ such that the following diagram commutes:
We call this function $\varphi$ a function preserving map.
The point-preimage $f^{-1}(a)$ is called a levelset or a fiber of our scalar field. More specifically, we can call $f^{-1}(a)$ an $a$-fiber of $f$. Since $\varphi$ is a function preserving map, it must carry $a$-fibers of $f$ to $a$-fibers of $g$. In other words, it "preserves levelsets".
We will now restrict this category to a subcategory of scalar fields where each of these new scalar fields are well-behaved. Let $S = \{a_1,\ldots,a_n\}$ be a finite set of real numbers, called "critical points". Then, we define a critical fiber, $\mathbb{V}_i$, for each $0 \leq i \leq n$ and a non-critical fiber, $\mathbb{E}_i$, for each $0 \leq i \leq n-1$. Each critical and non-critical fiber is defined to be a locally path-connected, compact space.
Now we define continuous attaching maps $l_i: \mathbb{E}_i \to \mathbb{V}_i$ and $r_i: \mathbb{E}_i \to \mathbb{V}_i$ for each $0 \leq i \leq n-1$. Finally, we let $X$ be the quotient space obtained by the disjoint union of $\mathbb{V}_i$ and $\mathbb{E}_i \times [a_i,a_i+1]$ by making the identifiactions $(l_i(x),a_i) \sim (x,a_i)$ and $(r_i(x),a_{i+1}) \sim (x,a_i+1)$ for all $i$ and for all $x \in \mathbb{E}_i$.
The intuition here is that the each non-critical fiber has a way of mapping to the critical fibers on either side in a continuous way. Now that we have a topological space $X$, we define the function $f$ as being a projection onto the second coordinate, making the pair $(X,f)$ a scalar field.
Any scalar field that is isomorphic to scalar field constructed in this way is called a constructible scalar field. The category of constructible scalar fields, denoted as $\mathbb{R}-\textbf{Top}^{\textbf{C}}$, is a subcategory of $\mathbb{R}-\textbf{Top}$, implying that morphisms in $\mathbb{R}-\textbf{Top}^{\textbf{C}}$ are the same as morphisms in $\mathbb{R}-\textbf{Top}$
Let's consider the example where our topological space is a 2-manifold $\mathbb{M}$ that is compact and differentiable at every point, and that the function $f$ is a morse function. We can think of this as being a hilly landscape, where the manifold is the base and the height of the peaks tell us what $f$ does to each point of the manifold. Furthemore, suppose there are no critical points that are shared. On the intervals between critical points, we select a single $a$-fiber, $a \in (a_i,a_{i+1})$ for every $0 \leq i \leq n-1$ and call it $\mathbb{E}_i$, the non-critical fiber of $i$. Now, for every critical point $a_i$ we look at the critical $a_i$-fibers and label them as $\mathbb{V}_i$. Critical points in this case are exactly minima, maxima, and saddles. The Category of Reeb Graphs and The Reeb Functor
Geometrically, there is a notion of what a Reeb graph of a scalar field is. It is a topological structure which preserves changes in path-connected components as we move across (or up) a scalar field, while removing features such as the width and shape of these path connected components. We can picture this, as we have done before, with a hilly landscape that is bounded and compact. We can imagine water filling up the landscape, eventually submerging all the of the peaks. As the water levels rise, different peaks become submerged, changing the topology of that specific landscape. At a single water level, there are only certain pieces of the landsacpe that will not be submerged. Standing on one peak, there might be no way to go to another peak without traversing the water, making this a separate path connected component. As the water rises, bridges that connect peaks get submerged and we lose the path-connectedness. The reeb graph attempts to track this change in topology.
Given a scalar field $(X,f)$, we can construct its Reeb graph as follows: We first define an equivalence relation between points $x,y \in X$ by stating that $x\sim y$ if $f(x) = f(y)$ (same "height" in our scenario) and only if $x,y$ are in the same path-connected component (accessible from one another without traversing water in our scenario). Then we can define the quotient space $X_f := X/\sim_f$ using this equivalence relation. We then define a new function $\tilde{f}$ as satisfying $\tilde{f}\circ \rho = f$, where $\rho$ is the quotient map $\rho : X \to X_f$. The pair $(X_f, \tilde{f})$ is the Reeb Graph of the scalar field $(X,f)$.
We can construct a new category called $\textbf{Reeb}$ which is the category of Reeb graphs. If we separate the notion that a Reeb graph is constructed from a scalar field, we can still provide a rigorous definition for what an object in $\textbf{Reeb}$ is. Formally, we define the category $\textbf{Reeb}$ to be the subcategory of $\mathbb{R}-\textbf{Top}^{\textbf{C}}$ such that all critical and non-critical fibers are 0-dimensional. There is a ton of stuff you already wrote about this that you can grab from above Because this is a subcategory, morphisms in $\textbf{Reeb}$ are the same as morphisms in $\mathbb{R}-\textbf{Top}^{\textbf{C}}$, which are continuous, function-preserving maps. We want to show that given $(X,f)$ is an object in $\mathbb{R}-\textbf{Top}^{\textbf{C}}$ that its Reeb graph $(X_f,\tilde{f})$ is an object in $\textbf{Reeb}$. The 0-dimensional fiber must be true because we know that an $a$-fiber in $(X,f)$ is a locally path-connected, compact space. So, the corresponding $a$-fiber in $(X_f,\tilde{f})$ is the contraction of each of the path-connected pieces to single point, meaning they are 0-dimensional (and they will still be locally path-connected and compact). Now we need to show that $\tilde{f}$ is a function preserving and a continuous map.
Munkres Theorem 22.2 Suppose $\rho$ is a quotient map $\rho : X \to Y$. Now suppose there exists another topological space $Z$ and a map $f: Y \to Z$ such that $f$ is constant on all subsets $\rho^{-1}(\{y\})$ for all $y \in Y$. Then there exists a function $f$ such taht $g \circ \rho = f$ and $g$ in continuous if and only if $f$ is continuous.
proof: We only need the backwards direction for this work. That is, we need to show that the function $f$ exists and that if $g$ is continuous, $f$ is continuous. Assume that $g$ is constant on point pre-images $\rho^{-1}(\{y\})$ and that $g$ is continuous. Then we know that $f(\rho^{-1}(\{y\}))$ is a single point in $Z$. Denote this point as $f(y)$. Then we have a map $g:X\to Z$ such that for all $x in X$, $(g \circ \rho)(x) = f(x)$. Now, let $V$ be and open set in $Z$. Then $f^{-1}(V)$ is open and $f^{-1}(V) = \rho(g^{-1}(V))$. Since $\rho$ is a quotient map (which means it must be continuous), $\rho^{-1}(U)$ is open when $U$ is open. Thus, $g^{-1}(V)$ must be open which implies that $g$ is continuous.
In our scenario, the quotient map is $\rho : X \to X_f$ and our continuous function is $f: X \to \mathbb{R}$. Since $f$ is constant on point pre-images of $X_f$ (by definition of the quotient space $X_f$), there exists a continuous function $g = \tilde{f}$ such that $\tilde{f} \circ \rho = f$. Thus, $\tilde{f}$ is a function-preserving, continous map which shows that $(X_f,\tilde{f})$ is in fact an object in the category $\textbf{Reeb}$.

We are now ready to define the Reeb Functor $\mathcal{R}: \mathbb{R}-\textbf{Top}^{\textbf{C}} \to \textbf{Reeb}$. Recall that a functor $F:\textbf{C} \to \textbf{D}$ is defined as a mapping between categories such that for each obect $A \in \textbf{C}$ there exists an object $F(A)$ and for each morphism $\alpha$ in $\textbf{C}$, there exists a morphism $F[\alpha]$. Furthermore, a functor must respect the identity element and composition. That is, $F[\alpha \circ \beta] = F[\alpha] \circ F[\beta]$ and $F[\text{id}_A] = \text{id}_{F(A)}$.
We start by defining $\mathcal{R}(X,f) = (X_f,\tilde{f})$. Then, we need to define a formula for how this functor maps a morphism in $\mathbb{R} - \textbf{Top}^{\textbf{C}}$ to a morphism in $\bar{\alpha}$ in $\textbf{Reeb}$. Let $\alpha: (X,f) \to (Y,g)$ be a morphism in $\mathbb{R} - \textbf{Top}^{\textbf{C}}$ and let $\rho_g : (Y,g) \to (Y_g,\tilde{g})$ and $\rho_f : (X,f) \to (X_f,\tilde{f})$ be quotient maps as defined above. Since $\alpha$ preserves level sets and is continuous (by definition of a morphism in this category), we know that the composition $\rho_g \circ \alpha$ is constant on $\rho_f({z})$ for all $z \in (X_f,\tilde{f})$ . To see this, suppose $x,y$ are in $\rho_f^{-1}({z})$ for some $z \in (X_f,\tilde{f})$. This implies that $x,y$ are in the same $a$-fiber of $X$ and they lie on the same path-connected component. Then, $\alpha(x)$ and $\alpha(y)$ are in the same $a$-fiber of $Y$ and also lie on the same path-connected component because continuous functions carry path-connected sets to path-connected sets. Then $\rho_g(\alpha(x)) = \rho_g(\alpha(y))$ which implies that this composite map is constant over elements of $\rho_f^{-1}({z})$. Cosheaves
Let $\textbf{Int}$ be the category whose objects are open intervals $I \subseteq \mathbb{R}$ and whose morphisms are inclusions $I \subseteq J$. Let the category $\textbf{Set}$ be the category whose objects are sets and whose morphisms are total functions (functions that use ever element in the domain set) from on set to the other. Then, we define the category $\textbf{Pre} = \textbf{Set}^{\textbf{Int}}$ as being the category of functors $\textbf{Int} \rightarrow \textbf{Set}$. This category is called the category of pre-cosheaves.
In other works, similar structures have been created and refered to as "indexed diagrams". For example, we could have the category $\textbf{Top}^{(\mathbb{R},\leq)}$ which is the category of functors $(\mathbb{R},\leq) \rightarrow \textbf{Top}$ where $(R,\leq)$ is the category of elements of the real line whose morphisms are ordering, and $\textbf{Top}$ is the category of topological spaces. While seemingly strange, we can interpret this as topological spaces that are "indexed" by values on the real line. When we think about persistent homology, we think of homology groups that are dependent on some real number (the birth time for a persistence diagram).
We can also talk about other pre-cosheaves, which are just depending on the target category of the functor. For example, we can have the category of pre-cosheaves $\textbf{Pre(Top)}$ which are functors $\textbf{Int} \rightarrow \textbf{Top}$. Then, we can do a post composition of this functor to move the topological spaces to sets. In this work, we consider the elements of $\textbf{Pre}$ to be functors defined as $F(I) = \pi_0(f^{-1}(I))$, where $\pi_0$ is the set of path connected components of the topological space, and $f$ is the function defined on our scalar field. So, we look at the connected components of the pull back of our scalar field.
Now, we want to convert our Reeb Graphs into these (pre)cosheafs. We define the Reeb Cosheaf functor $\mathcal{C}$ as follows: Let $f = (X,f)$ be a scalar field (technically, in our construction, this can just be the Reeb Graph itself, which is what's desired). Then, $\mathcal{C}(f) = F$ is the pre-cosheaf defined by \[F(I) = \pi_0(f^{-1}(I)), \hspace{15px} F[I\subseteq J] = \pi_0[f^{-1}(I) \subseteq f^{-1}(J)]\] While the definition of cosheaf is pretty simple, it is seemingly useless without a nice geometric realization. If we don't understand how the cosheafs are mapped to each other, we won't know how the Reeb Graphs are being compared. Recall that any morphism of scalar fields $\alpha : (X,f) \rightarrow (Y,g)$ is function preserving, meaning it preserves level sets. So, $\alpha$ automatically maps $f^{-1}(I)$ onto $g^{-1}(I)$. This is extremely important for computation, as well.
Recall that a natural transformation between two functors is a family of maps $\eta_X : F(X) \rightarrow G(X)$ where $X \in \textbf{C}$. Furthermore, it must be true that $\eta_Y \circ F(f) = G(f) \circ \eta_X$ where $f$ maps $X$ to $Y$ (which implies $F(f)$ maps $F(X)$ to $F(Y)$ and $G(f)$ maps $G(X)$ to $G(Y)$) and $\eta_Y$ is the mapping from $F(Y)$ to $G(Y)$. This composition property is referred to as "natural". This can be drawn with a commutative diagram easily Our map $\alpha$ is completely independent of the choice of I. Thus, we can create a natrual transformation between pre-cosheafs $F,G$ by defining the family of maps between $F,G$ to be $\mathcal{C}[\alpha]_I = \pi_0[f^{-1}(I) \rightarrow g^{-1}(I)]$. That is, for every choice of $I$, we already have a map $\alpha$ between $f^{-1}(I)$ and $g^{-1}(I)$. So, applying $\pi_0$ to the map (meaning the map now acts from $\pi_0(f^{-1}(I))$ to $\pi_0(g^{-1}(I))$) moves it to acting on cosheafs. But in this, case, it is now a family of maps (becasue it works for every interval $I$ in the initial category).
Interleaving Distance
Recall that a natural transformation between two functors is a family of maps $\eta_X : F(X) \rightarrow G(X)$ where $X \in \textbf{C}$. Furthermore, it must be true that $\eta_Y \circ F(f) = G(f) \circ \eta_X$ where $f$ maps $X$ to $Y$ (which implies $F(f)$ maps $F(X)$ to $F(Y)$ and $G(f)$ maps $G(X)$ to $G(Y)$) and $\eta_Y$ is the mapping from $F(Y)$ to $G(Y)$. This composition property is referred to as "natural". Let $F,G: \textbf{Int} \rightarrow \textbf{Set}$ be pre-cosheaves. An isomorphism between two functors consists of a family of maps \[\varphi_I: F(I) \rightarrow G(I) \hspace{10px} \text{and} \hspace{10px} \psi_I: G(I) \rightarrow F(I)\] for each $I \subseteq \mathbb{R}$, such that $\varphi_I$ and $\psi_I$ are inverses for each $I$ and are natural with respect to the inclusions $I\subseteq J$. That is, $\varphi_J \circ F[I\subseteq J] = G[I \subseteq J] \circ \varphi_I$.
If $I = (a,b)$, then let $I^{\varepsilon} = (a - \varepsilon, b + \varepsilon)$. An $\varepsilon$ - interleaving between $F,G$ is given by two families of maps \[\varphi_I: F(I) \rightarrow G(I^{\epsilon}) \hspace{10px} \text{and} \hspace{10px} \psi_I: G(I) \rightarrow F(I^{\epsilon})\] that are natural with respect ot inclusions $I \subseteq J$ and such that \[\psi_{I^{\varepsilon}} \circ \varphi_I = F[I \subseteq I^{2\varepsilon}] \hspace{10px} \text{and} \hspace{10px} \varphi_{I^{\varepsilon}} \circ \psi_I = G[I \subseteq I^{2\varepsilon}]\] When $\varepsilon = 0$, then this is precisely an isomorphism between $F,G$.
If we go back to our canonical example, we have a Reeb Graph constructed from our scalar field. At every interval of the real line, we have some pull back on the actual topological space. For two different scalar fields that we are comparing, $f,g$, we are always looking within the same "window". What we seek to find is the value of $\epsilon$ such that mapping from any pullback of $f$ to the pullback of a "thickened" $g$.
Computational Methods
To be able to compute any of these structures, we first need to introduce some preliminary defitinitions for the spaces that we compute on.
A $n$-simplex is the convex hull of $n+1$ geometrically independent (affinely independent) points embedded in $\mathbb{R}^n$.
A finite simplicial complex $\mathcal{K}$ in $\mathbb{R}^N$ is a finite collection of simplices such that every face of a simplex $\sigma$ is also in $\mathcal{K}$ and the intersection of two simplices in $\mathcal{K}$ is a face of both of them (or neither of them). We will only refer to finite simplicial complexes in this work (as opposed to infinite simplicial complexes) so we will drop the term "finite" from now on.
The underlying space $|\mathcal{K}|$ of a simplicial complex $\mathcal{K}$ is the union all simplices in $\mathcal{K}$ as a space embedded in $\mathbb{R}^N$. This definition drives at the difference between the combinatorial nature of a simplicial complex $\mathcal{K}$ (sometimes called an abstract simplicial complex) in comparison with the actual points it contains when realized in $\mathbb{R}^N$. Thankfully, we can store just the abstract simplicial complex in memory and pull the underlying space from it as we wish.
Suppose we have a topological space $\mathbb{X}$. The triangulation $\mathcal{T}$ of $\mathbb{X}$ is a simpilicial complex $\mathcal{K}$ whose underlying space $|\mathcal{K}|$ is homeomorphic to $\mathbb{X}$. Note that in this construction, the underlying space $|\mathcal{K}|$ would relate to the actual physical representation of the space $\mathbb{X}$, while the simplicial complex $\mathcal{K}$ is simply the combinatorial representation of this space.
The data sets that we often compute on are pulled from some manifold $\mathbb{M}$. Of course the data cannot be sampled at every point of the manifold, so we introduce the concept of a piecewise-linear manifold which is the triangulation of a manifold $\mathbb{M}$. Thus, a piecweise-linear manifold is a combinatorial representation of the manifold. From this, we can create a piecweise-linear scalar field
Let $\mathbb{M}$ be a compact manifold and let $f:\mathbb{M} \to \mathbb{R}$ be a continuous scalar function. Then the pair $(\mathbb{M},f)$ is an object in the category $\mathbb{R}-\textbf{Top}$ hopefully constructible. From this we can create a piecewise linear manifold by taking the triangulation of the manifold $\mathbb{M}$. This induces a new function $\hat{f}$ which is defined as $\hat{f}(x) = \sum^d_{i=0} \alpha_i f(\tau^i_0)$, where $\tau^i_0$ are the 0-dimensional simplices (vertices) of the simplex $\sigma$ such that $x \in \sigma$ and the coefficients $\alpha_i$ are the barycentric coordinates of $x$.
Suppose we have two scalar fields $f$ and $g$ such that they are defined on the same domain (so their scalar functions are the only things discerning them). Now suppose that there exists no point $x \in X$ such that $g(x) = a$ but there exists some subset $U \subseteq X$ such that $f(x) = a$ for all $x\in U$. Then, the $a$-fiber of $f$ is $U$ while the $a$-fiber of $g$ is the empty set. So how does $phi$ work in this case? If there is multiple values of $\mathbb{R}$ such that this situation arises, we run into the problem of $\varphi$ mapping from $U_1$ to the empty set and $U_2$ mapping to the empty set as well, which would break commutativity because $g$ on the empty set cannot map to two differennnt values. Note that this situation can arise even if the domains differ, of course.
